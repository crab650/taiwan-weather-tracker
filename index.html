請按照以下步驟完成設定，以啟用氣象面板：

1
取得 API 授權碼：前往 氣象資料開放平臺 註冊會員並在「個人會員專區」複製您的 API 授權碼。
2
修改設定檔：打開專案目錄下的 config.json，將 cwa_api_key 的值修改為您的 API 授權碼。
3
抓取資料：在終端機執行 python weather_tracker.py --fetch 進行首次抓取。
